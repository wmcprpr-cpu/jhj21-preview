Project · Public Policy · Social Innovation · Global Collaboration

연구를 사회적 실천으로 연결하는 프로젝트

집현전21의 프로젝트는 연구 성과를 현실과 연결하는 실행의 영역입니다. 공공정책 프로젝트, 사회혁신 프로젝트, 국제협력 프로젝트를 통해 사회 문제 해결과 공공의 가치 확산을 위한 구체적인 실천 모델을 제안하고 추진합니다.

공공정책 프로젝트 사회혁신 프로젝트 국제협력 프로젝트

실행 지향

연구의 실천적 확장

프로젝트는 연구 결과를 제안에 그치지 않고 실제 실행 가능한 모델로 연결하는 과정입니다.

핵심 축

정책 · 혁신 · 국제협력

집현전21은 세 가지 프로젝트 축을 통해 사회와 지식을 실제 변화로 이어갑니다.

기대 효과

사회문제 해결과 공공 가치 확산

프로젝트는 사회적 수요를 반영하고 협력적 실행 구조를 통해 지속 가능한 효과를 추구합니다.

공공정책 프로젝트 정책 실행과 제도 개선 사회혁신 프로젝트 사회 문제 해결형 실천 국제협력 프로젝트 국내외 협력과 확장
Public Policy Project

공공정책 프로젝트

공공정책 프로젝트는 연구 결과를 토대로 공공 정책 개선과 제도적 실행 가능성을 높이기 위한 프로젝트입니다. 사회 문제 해결을 위한 현실적 대안을 모색하고 정책적 실행 모델을 제안합니다.

Policy Design

정책 설계

정책 연구 결과를 토대로 실행 가능한 공공정책 모델과 제도 개선 방향을 설계합니다.

Implementation

실행 모델

정책이 실제 사회 현장에서 작동할 수 있도록 단계별 실행 모델과 운영 구조를 제시합니다.

Public Value

공공성 강화

공공의 이익을 최우선으로 하여 제도 개선과 사회적 효용을 높이는 프로젝트를 추진합니다.

Social Innovation Project

사회혁신 프로젝트

사회혁신 프로젝트는 복합적인 사회 문제를 해결하기 위한 실천적이고 협력적인 프로젝트입니다. 연구와 현장을 연결하여 새로운 문제 해결 방식을 모색합니다.

Problem Solving

사회 문제 해결

가족, 공동체, 문화, 환경, 건강 등 다양한 사회 문제에 대한 실질적인 해법을 탐색합니다.

Innovation Model

혁신 모델 개발

지식 네트워크와 협력을 통해 새로운 사회혁신 모델과 공익적 실천 구조를 설계합니다.

Community Link

현장 연계

연구 결과가 현장과 지역사회에 연결될 수 있도록 실제 적용 가능성을 중심으로 추진합니다.

International Collaboration Project

국제협력 프로젝트

국제협력 프로젝트는 국내외 연구기관과 협력 단체, 전문가 네트워크와의 연계를 통해 지식과 연구 성과를 글로벌 수준으로 확장하는 프로젝트입니다.

Global Network

국내외 협력 네트워크

연구기관, 대학, 전문가 그룹, 협력 단체와의 연계를 통해 국제적 협력 기반을 마련합니다.

Knowledge Exchange

지식 교류 확대

국제적 연구 교류와 협력 프로젝트를 통해 집현전21의 지식 기반과 활동 영역을 확장합니다.

Future Expansion

글로벌 확장성

한국 사회의 연구 성과와 공공 가치를 세계와 연결하며 국제협력 모델을 발전시킵니다.

Project Principles

프로젝트 운영 원칙

집현전21의 모든 프로젝트는 공공성, 협력성, 지속가능성을 바탕으로 기획되고 운영됩니다.

Principle 01

공공성

모든 프로젝트는 공공의 이익과 사회적 가치 실현을 중심에 둡니다.

Principle 02

협력성

연구위원, 자문위원, 외부 전문가, 기관과의 협력을 통해 프로젝트를 추진합니다.

Principle 03

실천성

연구 결과가 실제 정책과 사회 현장에 연결될 수 있도록 실행 가능성을 높입니다.

Principle 04

지속가능성

단기 성과를 넘어 장기적 사회 효과와 지속 가능한 운영을 지향합니다.

Project Flow

프로젝트 추진 흐름

집현전21의 프로젝트는 연구를 기반으로 기획되고, 협력 구조를 통해 실천 단계로 확장됩니다.

Step 01

문제 진단

사회 문제와 정책 수요를 분석하고 핵심 과제를 도출합니다.

Step 02

연구 기반 마련

정책연구와 학술연구를 통해 프로젝트의 근거와 방향을 정리합니다.

Step 03

협력 실행

연구위원, 기관, 현장 파트너와 함께 프로젝트를 구체화하고 추진합니다.

Step 04

성과 확산

프로젝트 결과를 포럼, 발간물, 교육, 정책 제안으로 확장합니다.
